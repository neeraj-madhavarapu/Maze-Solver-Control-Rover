# Maze-Solver-Control-Rover
Maze Solver Control Rover is an Arduino-based autonomous robot that navigates a maze using differential drive kinematics, IR and ultrasonic sensors, obstacle avoidance, and the A* path-planning algorithm. It autonomously finds the shortest path to the destination by integrating embedded control, sensor processing, and intelligent navigation.

Led a team of four in designing and building an autonomous maze-solving rover from scratch using off-the-shelf components. Developed Arduino-based firmware for precise motion control by implementing differential drive kinematics. Integrated obstacle detection and avoidance using IR and ultrasonic sensors, and implemented the A* path-planning algorithm to enable the rover to navigate a maze and reach the destination through the shortest possible path. The project provided hands-on experience in embedded systems, robotics, sensor integration, and autonomous navigation.
